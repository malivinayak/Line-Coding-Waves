Line Coding Waves

Line encoding is the process of converting digital data into analog signals suitable for transmission over communication channels.

We will be generating waveforms for Unipolar NRZ, Polar NRZ-L, Polar NRZ-I, Polar RZ, Manchester, Differential Manchester, AMI, and Pseudoternary line encoding techniques.

Wave form are as follows
Unipolar NRZ Line wave
Polar NRZ-L Line wave
Polar NRZ-I Line wave
Polar RZ Line wave
Manchester Line wave
Diff-Manchester Line wave
AMI Line wave
Pseudoternary Line wave
